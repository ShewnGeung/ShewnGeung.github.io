CentOS服务器下配置java环境

发表于2025-01-23|更新于2025-01-23|java
|字数总计:260|阅读时长:1分钟

由于服务器只需要运行环境而不需要开发环境，我们只需要在服务器安装 JRE 即可。相对于 JDK 而言，JRE 的压缩包体积更小，便于更快上传。

下载JRE

本文使用Liberica JDK/JRE，版本为jre-8u442。

  1. 选择TAR.GZ版本下载。
  2. 将下载好的JRE上传到服务器。

安装JRE

  1. 使用tar命令解压压缩包。
  2. 打开配置文件。
  3. 按下 i 键进入编辑模式，在文件末尾追加以下内容，然后按下 ESC 退出编辑模式，输入:wq保存并退出文件。
  4. 重新加载配置文件，使环境变量生效。
  5. 输入java -version查看Java版本，显示如下内容即为配置成功。
